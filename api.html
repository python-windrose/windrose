bar(direction, var, **kwargs)ÔÉÅ

Plot a windrose in bar mode. For each var bins and for each sector, a colored bar will be draw on the axes.

Parameters

  • direction (1D array) ‚Äì directions the wind blows from, North centred

  • var (1D array) ‚Äì values of the variable to compute. Typically the wind speeds.

  • nsector (integer, optional) ‚Äì number of sectors used to compute the windrose table. If not set, nsectors=16, then each sector will be 360/16=22.5¬∞, and the resulting computed table will be aligned with the cardinals points.

  • bins (1D array or integer, optional) ‚Äì number of bins, or a sequence of bins variable. If not set, bins=6 between min(var) and max(var).

  • blowto (bool, optional.) ‚Äì if True, the windrose will be pi rotated, to show where the wind blow to (useful for pollutant rose).

  • colors (string or tuple, optional) ‚Äì one string color (‚Äòk‚Äô or ‚Äòblack‚Äô), in this case all bins will be plotted in this color; a tuple of matplotlib color args (string, float, rgb, etc), different levels will be plotted in different colors in the order specified.

  • cmap (a cm Colormap instance from matplotlib.cm, optional.) ‚Äì if cmap == None and colors == None, a default Colormap is used.

  • edgecolor (string, optional) ‚Äì The string color each edge box will be plotted. Default : no edgecolor

  • opening (float, optional) ‚Äì between 0.0 and 1.0, to control the space between each sector (1.0 for no space)

  • calm_limit (float, optional) ‚Äì Calm limit for the var parameter. If not None, a centered red circle will be draw for representing the calms occurrences and all data below this value will be removed from the computation.
